참고사항
=========

본 소프트웨어는 LGPL(Lesser General Public License) 라이센스를 따르는 라이브러리인 Qt Module 및 SIMDIS SDK 를 사용합니다. 
이는 다음과 같은 사항을 명시합니다.

- 안내 사항 : 
    SIMDIS SDK 및 Qt Module을 사용하고 있으며, 해당 라이브러리에 대한 사용 및 배포는 LGPL 라이선스를 준수해야 합니다.
    (https://www.gnu.org/licenses/lgpl-3.0.html)

- 보증책임 부인 :
   Qt Module 및 SIMDIS SDK 사용으로 인한 어떠한 명시적이거나 묵시적인 보증도 포함되지 않습니다. 
   이는 이 라이브러리를 사용함으로써 발생하는 모든 리스크는 사용자에게 있음을 의미합니다.

- 라이선스 사본 제공 : 
   이 소프트웨어를 양도받는 모든 사용자에게는 소프트웨어와 함께 LGPL 라이선스의 사본(LICENSE.txt)파일로 제공 합니다. 
   이를 통해 사용자들은 LGPL 라이선스의 내용을 숙지하고 준수해야 합니다.

- 사용 조건 : 
    LGPL 라이선스에 따라 이 소프트웨어를 수정하거나 재배포할 경우, 해당 작업물의 소스 코드를 공개해야 합니다.

- 라이브러리 형태로의 수정 :
   소프트웨어를 라이브러리 형태로 수정하여 사용하는 것이 허용 되며, 라이브러리 형태로 수정한 경우 수정된 부분과 수정된 날짜를 소스 코드 또는 해당 파일에 명시해야 합니다.
   이는 소프트웨어를 기반으로 새로운 라이브러리를 만들어 사용할 수 있음을 의미합니다.

- 원본 및 파생 저작물의 소스 코드 제공 :
   소프트웨어의 원본 및 파생 저작물을 사용, 수정, 배포하는 경우, 해당 소프트웨어의 소스 코드를 제공해야 합니다. 
   사용자가 소스 코드 제공을 요청할 경우, 해당 요청을 받아들이고 적절한 방법으로 소스 코드를 제공해야 합니다.
   이는 소스 코드를 요청하는 사용자에게 소프트웨어의 소스 코드를 제공할 의무가 있음을 의미합니다.

- 응용 프로그램 사용 허용 :
   수정된 라이브러리를 사용하여 만든 응용 프로그램을 사용할 수 있으며 이는 사용자가 수정된 라이브러리를 사용하여 자유롭게 응용 프로그램을 개발하고 배포할 수 있음을 의미합니다.

- 설치 정보 제공 :
   사용자가 제품을 설치할 때 사용된 LGPL 라이브러리의 목록과 해당 라이브러리가 어디에서 다운로드할 수 있는지 등의 정보를 제공해야 합니다.
   설치 정보에는 사용된 각 라이브러리의 저작권 정보와 LGPL 라이선스의 내용을 포함해야 합니다. 


소프트웨어 소개
================

본 소프트웨어는 SIMDIS SDK 활용하여 3D 가시화 소프트웨어를 개발하기 위한 테스트용 소프트웨어 입니다.
MSCD/SMSS/RIPS는 IMT에서 개발되었습니다.


소프트웨어 지원
===============
IMT 홈페이지를 통해 (www.imtplus.com) 제한적으로 무료 지원을 받으실 수 있습니다. 
도움을 받으려면 연락 부탁드립니다.
