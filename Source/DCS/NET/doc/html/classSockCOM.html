Detailed Description

Cross-platform library for reading and writing to COM ports. Can read and write to COM 1-n.

Create a new SockCOM object bound to the specific port. The device name is determind by appending the specified port number to the following strings: 'COM' + comPort for the Microsoft Windows operating system '/dev/ttyS' + comPort for Linux systems '/dev/term/' + (comPort + 'a') for SunOS systems '/dev/ttyf' + comPort for other UNIX systems
