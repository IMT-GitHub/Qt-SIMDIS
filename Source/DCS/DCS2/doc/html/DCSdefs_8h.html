enum DCSCoordSystem_t

Supported coordinate system types.

Identifiers for coordinate systems recognized by DCS and SIMDIS. The position informationed contained in a DCS data packet are expected to be represented by the specified coordinate system type.

Enumerator:
DCSCOORD_WGS84 

Earth Centered, Earth Fixed Geocentric coordinate system: based on WGS-84.

DCSCOORD_LLA 

Geodetic coordinate system with position information specified as x=lat,y=lon,z=alt.

DCSCOORD_NED 

Topographic coordinate system with position information specified as x=north,y=east,z=down.

DCSCOORD_NWU 

Topographic coordinate system with position information specified as x=north,y=west,z=up.

DCSCOORD_ENU 

Topographic coordinate system with position information specified as x=east,y=north,z=up.

DCSCOORD_XEAST 

Cartesian tangent plane system with position information specified as x=east,y=north,z=up.

DCSCOORD_GTP 

Flat Earth: Generic Tangent Plane, User defined X-Y rotation and tangential offset.

DCSCOORD_ECI 

Earth Centered Inertial Geocentric coordinate system: based on WGS-84.

enum DCSRequestType_t

DCS client data requests.

Valid data requests that a client may make from the server.

Enumerator:
DCSREQUEST_CONNECT 

Client requests negotiation of a new connection.

DCSREQUEST_VERSION 

Client requests DCS API version.

DCSREQUEST_DATAMODE 

Client requests the type of UDP socket used for data transmission.

See also:
Data Modes.
DCSREQUEST_TIME 

Client requests time data in the form of DCSTimeHeader.

DCSREQUEST_SCENARIO 

Client requests scenario data in the form of DCSScenarioHeader.

DCSREQUEST_EVENT 

Server sends new event to all connected clients. Not to be used by client for making requests.

DCSREQUEST_HEADER 

Server sends new header to all connected clients. Client requests a header for a specific id. Also used for sending DCSGenericData and DCSCategoryData over TCP

DCSREQUEST_HEADERS 

Client requests headers for all known objects. Header objects are DCSPlatformHeader, DCSBeamHeader, DCSGateHeader.

DCSREQUEST_DISCONNECT 

Client requests to disconnect from server or is ordered to disconnect by server.

DCSREQUEST_KEEPALIVE 

Client/server check for active connection.
