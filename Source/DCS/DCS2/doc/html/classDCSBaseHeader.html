Detailed Description

An abstract base class for DCS network header data types.

All DCS header data types to be sent to the network are derived from this class. Headers are sepearted from general data because they require special processing for storage and client server connection negotiations. The main purpose of using a base header class is to limit the types of data that can be stored in the DCSHeaderContainer.
