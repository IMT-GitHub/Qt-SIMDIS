Detailed Description

A top level class for implementing a DCS data server.

The DCSServer class facilitates the creation, setup and sending of data via the NRL developed DCS protocol

Parameters:
p_appNamea string representing the name of the data server.
protoa char representing the type of UDP socket: broadcast, mulitcast, or unicast, to use. See DCSdefs.h for more information.
srvPortan integer representing the client's connection port number to the server.
dataPortan integer representing the port number DCS data is sent.
p_dataAddra string representing the IP address DCS data is sent.
p_ifacea string representing the network address of the network interface to use for data transmission. To use any network interface (INADDR_ANY), set to NULL.
ttlan integer specifying the new time to live value to be used (Multicast only).

The DCSServer class facilitates the creation, setup and sending of data via the NRL developed DCS protocol
