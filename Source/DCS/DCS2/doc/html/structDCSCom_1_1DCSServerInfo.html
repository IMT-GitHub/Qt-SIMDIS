char DCSCom::DCSServerInfo::server_id[DCS_SERVERIDSZ]

Unique server ID.

Unique ID to be used for filtering data sent to clients. With this, multiple servers can send data on the same address, and the client can ignore any data from the unknown servers, by only processing data that has the proper server_id value.

ClientList DCSCom::DCSServerInfo::vagrants [mutable]

TCP closed connection list.

List of TCP client sockets whose connections have been closed. If one of the server functions detects a client connection has closed it will add a reference for that socket to this list so that the application may properly dispose of the client sockets.
