DCS Reference Documentation

Introduction

The DCS API is a C++ software library developed by the Tactical Electronic Warfare Division at the Naval Research Laboratory. The API is the software implementation of a system for sending data to SIMDIS for display, and is distributed as part of the SIMDIS 3D Visualization and Display package.

This document is the DCS API reference. It contains reference documentation for all components of the DCS API, consisting of:

  • A list of all documented namespaces.
  • A documented class hierarchy.
  • An annotated list of all documented classes, structs, unions, and interfaces.
  • A list of all documented files.
  • A list of all documented namespace members.
  • A list of all documented class members.
  • A list of all documented global file members.

  • A simple DCSServer example
  • A simple client example
