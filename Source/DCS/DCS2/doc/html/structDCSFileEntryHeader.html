Detailed Description

A structure for tagging each record added to the data file.

Each record added to the data file shall be prepended with this structure. The strucutre serves the purpose if identifying the state of the record, active or marked for deletion, and indicating the size of the record, for reloading, external processing, etc. This structure is used internally by DCSHeaderContainer.
